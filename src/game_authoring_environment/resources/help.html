Creation panel

This is the top left panel. It allows the user to add and delete scenes and actors as well as define actor behavior.

Tab Label Functionality
Scene

The Add button adds a new Scene. Scenes are listed in the panel below.

The Delete button deletes the scene highlighted in the panel below.
Actors

The Add button adds a new actor. Actors are listed in the panel below.

The Delete button deletes the actor highlighted in the panel below.
Behaviors

The Enter Tile Mode button allows the user to create tiles by dragging on the visial display. Clicking this button pops up an dialog box in which the user may enter the collision ID.

On clicking OK, another dialog box opens up where the user can decide whether or not to select the brick image

Clicking ok on this dialog opens an image choosing dialog box

Back on the Behaviors panel, the Exit Tile Mode button disables tile creation mode.

Game editing panel

This is the bottom left panel. It allows the user to edit scenes and actors.

Tab Label Functionality
Game Under this tab the user can implement settings pertaining to the entire game such as player initial score etc.
Scene Under this tab the user may select a scene's background.
Actor Under this tab the user may select an actor's image and set attributes such as movement and shooting.
Player Under this tab the user may select the player's image and set attributes such as name and shooting.

Visual display

This is the bottom right panel. The user can see the game that they're editing here.
