Creation panel

This is the top left panel. It allows the user to add and delete scenes and actors as well as define actor behavior.

Tab Label Functionality
Scene

The Add button adds a new Scene. Scenes are listed in the panel below.

The Delete button deletes the scene highlighted in the panel below.
Actors

The Add button adds a new actor. Actors are listed in the panel below.

The Delete button deletes the actor highlighted in the panel below.

Game editing panel

This is the bottom left panel. It allows the user to edit scenes and actors.

Tab Label Functionality
Scene
Actors

Visual display

This is the bottom right panel. The user can see the game that they're editing here.
